Node to reach a goal autonomously.

Author
Sara Sgambato
Version
1.0
Date
01/05/2022

Subscribes to:
/move_base/feedback /move_base/status

Publishes to:
/move_base/goal /move_base/cancel

Description :

This node prompts the user to either set a new goal or cancel the current goal.

Function Documentation

◆ cancelGoal()

void cancelGoal ( )

Function to cancel the current goal.

The function checks whether a goal has been set via the variable "flag": if not, a message is printed warning the user that no goal has been set yet, otherwise the function cancels the current goal, sets the flag to false and informs the user that the operation succeded.

◆ choice()

char choice ( )

Function to prompt the user.

Returns
a char corresponding to the user input.

This function asks the user to set a new goal or to cancel the current one. The choice is saved in a variable of type char.

◆ getStatus()

void getStatus ( const actionlib_msgs::GoalStatusArray::ConstPtr &  msg)

Callback function for the /move_base/status topic subscriber.

Parameters
msgis the message of type GoalStatusArray.

This callback function is used to get the status of the current goal: if the goal is either succeded, aborted or rejected, then the function informs the user of the status and cancels the goal.

◆ setGoal()

void setGoal ( )

Function to set the goal.

This function asks the user to insert valid coordinates of a point to reach. After a valid input is detected, the goal parameters are set and the flag of the goal is set to true. The status is checked via the subscriber to the /move_base/status topic.
